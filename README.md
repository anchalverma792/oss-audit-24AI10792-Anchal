## 📌 Project Overview
This project presents a detailed analysis of **Visual Studio Code**, focusing on its role in the open-source software ecosystem. It explores its origin, licensing model, ethical considerations, Linux footprint, ecosystem, and comparison with proprietary tools.

---

## 🟣 1. Introduction
Open source software has become the foundation of modern computing, enabling collaboration, innovation, and knowledge sharing. Visual Studio Code is a lightweight and powerful code editor widely used by developers. This project analyzes its importance and impact in the open-source world.

---

## 🟣 2. Origin and Philosophy

### 🔹 Origin
- Developed by Microsoft (2015)  
- Designed as a lightweight alternative to heavy IDEs  
- Built using Electron for cross-platform support  

### 🔹 License
- Open-source core: Code-OSS (MIT License)  
- Official version includes proprietary components  
- Flexible licensing allows wide adoption  

### 🔹 Ethics
- Follows an open-core model  
- Balances community contribution and corporate control  
- Promotes transparency with sustainable development  

---

## 🟣 3. Linux Footprint
- Installed using package managers (apt, pacman, etc.)  
- Executable located in `/usr/bin`  
- Runs as a normal user process (no background service)  
- Lightweight and efficient  
- Regular updates via package manager  

---

## 🟣 4. FOSS Ecosystem
- Supports multiple languages via extensions  
- Built on Node.js and Electron  
- Integrates with Git and GitHub  
- Strong extension marketplace  
- Supported by global developer community  

---

## 🟣 5. Open Source vs Proprietary

| Feature | VS Code (Open Source) | Sublime Text (Proprietary) |
|--------|----------------------|-----------------------------|
| Cost | Free | Paid |
| Source | Open | Closed |
| Customization | High | Limited |
| Extensions | Large ecosystem | Smaller ecosystem |
| Performance | Moderate | Fast |
| Updates | Community-driven | Company-controlled |

## 🟣 6. Shell Scripts

### 🔹 Script 1 – System Identity Report
Displays system information such as kernel, user, uptime, and date.

### 🔹 Script 2 – Package Inspector
Checks if VS Code is installed and shows version and location.

### 🔹 Script 3 – Disk Auditor
Analyzes directories, permissions, and sizes.

### 🔹 Script 4 – Log Analyzer
Counts keyword occurrences in log files and displays matches.

### 🔹 Script 5 – Manifesto Generator
Creates a personalized open-source manifesto based on user input.
